# Legal CRM Dashboard

A modern, intuitive CRM system for lawyers to manage cases, documents, and client relationships with a clean, professional interface focused on efficiency and organization.

## Features

- 📊 **Dashboard Overview**
  - Case statistics and metrics
  - Recent activities feed
  - Upcoming deadlines calendar
  - Quick-access cards

- 📁 **Case Management**
  - Complete CRUD operations
  - Status tracking
  - Advanced filtering
  - Document attachments
  - Timeline view

- 👥 **Client Management**
  - Detailed client profiles
  - Case history
  - Client-case associations

- 📄 **Document Management**
  - PDF and image uploads
  - Document preview
  - Organized by case

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Shadcn/ui Components
- React Router

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

## Project Structure

```
src/
```
